== Description ==

Simple plugin to never write again Wordpress, Word Press or whatever. One word, capital W and P, it's **WordPress**!

For documentation and tips, please refer to the official plugin page for [Correctly Spell WordPress](http://planetozh.com/blog/my-projects/plugin-to-correctly-spell-wordpress-not-word-press/ "Correctly Spell WordPress")

== Installation ==

Installation is, as usual :

1. Upload files to your `/wp-content/plugins/` directory (preserve sub-directory structure if applicable)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Refer to the official plugin page for documentation, usage and tips
